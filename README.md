# leave-school
离校登记系统，为辅导员解决学生离校登记难题



后端采用spring boot编写，前端部分采用react编写，手机端页面采用sui编写

前台页面地址：http://127.0.0.1:8085
后台页面地址：http://127.0.0.1:8085/#/admin
